Greet-Tweet-Know by uberchicgeekchick

Greet-Tweet-Know is a a Twitter client for GTK+/GNOME focused on creating the most full featured Twitter client available any where.

Greet-Tweet-Know is a a Twitter client for GTK+/GNOME focused on creating the most full featured Twitter client available any where, not just on GNOME & Linux, even though the experience of using Greet-Tweet-Know. Greet-Tweet-Know goals are to provide access to all of Twitter's functions and to add interfaces that make using & managing your Twitter status, messages, friends, & followers. Greet-Tweet-Know can easily be embedded into other applications.

Install

For information on packages for specific Linux distributions please visit: http://uberchicgeekchick.com/?projects=greet-tweet-know&download=packages Otherwise Greet-Tweet-Know can be installed from source using the standard process for building applications. Start by extracting the tarball you've downloaded & open a terminal in, or cd to, the directory the tarball created. Than start by simply running: ./configure Note: on 64bit Linux distros you may need to help out configure a lil by running: ./configure --libdir=/usr/lib64 After you've run the above command simply run: make If after running make you don't have any errors than go ahead & install Greet-Tweet-Know. By default make will install Greet-Tweet-Know to /usr so you may need root privileges to finish installing Greet-Tweet-Know. if this is the case, which it usually is, than you'll need to run the following command: sudo make install If you don't need to have root privileges to install Greet-Tweet-Know than simply run: make install For more information on installing Greet-Tweet-Know please visit: http://uberchicgeekchick.com/?projects=greet-tweet-know
